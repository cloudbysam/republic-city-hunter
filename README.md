# Republic City Hunter RPG 

Welcome! This is a text-based Python game set in the world of *Avatar: The Legend of Korra*. You play as an Equalist operative working for Amon, hunting down rogue benders in the dark corners of Republic City. 

---

##  The Story Behind This Project

I am learning Python backend development! After studying some major coding concepts, I wanted to build a real project to put my new skills to the test. Instead of just doing boring textbook exercises, I built this text RPG inside a single, clean script to practice:

*   **Functions (`def`):** Breaking my game down into reusable blocks of code for different actions and areas.
*   **Decorators (`@`):** Building a custom action logger that wraps around my functions to track player actions cleanly.
*   **Error & Exception Handling:** Fixing tricky edge-case bugs (like checking if the inventory is empty before picking a random item, or handling player inputs safely) so the game never crashes.

---

##  Game Features

*   **Dynamic Encounters:** Fights and ambushes use random flavor text based on different bending styles.
*   **Notoriety System:** The more successful your raids, the more your notoriety increases.
*   **The Ultimate Boss Fight:** If your notoriety hits maximum (5/5), the game intercepts you and forces you to face **Avatar Korra** in a high-stakes showdown where you must choose to fight or surrender!

##  How to Run the Game

1. Clone this repository to your machine:
   ```bash
   git clone [https://github.com/cloudbysam/republic-city-hunter.git](https://github.com/cloudbysam/republic-city-hunter.git)
   ```
2. Open the project folder:   
    ```bash
   cd republic-city-hunter
   ```
3.  Run the script: 
     ```bash
    python main.py
    ```
